Example de fiche "SANS QUESTION"

    ```
    page "Ceci est le titre de la page"
    text "Ceci est un paragraphe de texte qui supporte le **markdown**."
    ```

Example de fiche "SANS QUESTION + url (ou url d'image)"

    ```
    page "Ceci est le titre de la page"
    text "Ceci est un texte"
    image "./Emplacement/de/limage/dans/le/même/dossier/que/le/fichier"
    subtext "Sous titre de l'image (légende), cliquer sur l'image l'ouvre dans son emplacement ou navigateur"
    text "Un autre parapgrahe en dessous"
    ```
Example de fiche "VRAI OU FAUX" (le true a la fine de tf lien vers la bonne réponse, ici VRAI)

    ```
    tf "Titre, question, tf = true or false", true
    text "plus de description"
    ```

Example de fiche "QCM" (mcq pour Multiple choice question, ici le B à la fin dit que la bonne réponse est la B)

    ```
    mcq "Question", B
    text "Texte"
    A "Réponse A"
    B "Réponse B"
    C "Réponse C"
    D "Réponse D"
    ```
